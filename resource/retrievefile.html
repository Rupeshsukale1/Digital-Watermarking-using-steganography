Retrieve File Help

For Retrieving the file follow the points given below :

  • Go in the Window menu.
  • Click on the Retrieve File menu.
  • Select the Input File (from which you want to retrieve the message) by browsing the system or enter the path of the intended file.
  • Click on Retrieve button to retrieve the message from the Input File.
  • After clicking on Retrieve button, the message will be displayed in the text area.
  • If the selected Input File do not contain any message then the system will show message "File Does Not Contain Any Message".
  • If the Input File contains the message, then click on save button. Select the location to save the message or enter the path for the location.
  • After selecting the location to save the file click on OK button.
  • Click on Reset button to reset or clear all the fields.
  • Click on close button to close the Retrieve File dialog box.
  • If text field is not given any input then system will show the message "Please Select input file for Retrieve operation.

After clicking the the retrieve button, file would be sent if all the information entered is right and user is online.
