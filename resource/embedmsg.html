Embed Message Help

For Embedding the message into file follow the points given below :

  • Go in the Window menu.
  • Click on the Embed Message menu.
  • Select the Input File by browsing the system or enter the path of the intended file.
  • Select the Output File by browsing the system or enter the path of the intended file.
  • Enter the Message in the text area.
  • Click on Embed button if you want to embed the message.
  • If any of the field is left vacant and user tries to embed the message, then it will show the message to user to enter the details.
  • Click on Reset button if you want to reset the or clear the fields entered by the user.
  • Click on Send button if you want to send the message to the receiver.
  • Click on Close button if you want to close the Embed Message dialog box.

If any of the field is left vacant then system will ask the user to give the proper input.
