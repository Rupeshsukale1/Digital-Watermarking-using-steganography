Send File Help

For Sending File follow the points given below :

  • Go in the Window menu.
  • Click on the Send file menu.
  • Enter the destination address.
  • Select the file (which is to be sent) by browsing the system or enter the path of the intended file.
  • Click the Send button.
  • If any of the field is kept vacant then, it will show message to enter the details.

After clicking the the send button file would be sent if all the information entered is right and user is online.
