Login Help

For Logging in into the system, follow the points given below :

  • Go in the File menu.
  • Click on the Login menu or press ctrl + l button.
  • Enter the User Name in the user name text field.
  • Enter the Password in password text field.
  • If the User Name and Password are proper then user can access the system.
  • If the User Name and Password in not proper the it will show the message "Invalid User Name And Password"
  • Click on Ok button after entering the details.
  • Click on Cancel button to cancel if you don't want to enter into the system and exit.

If the user is authorized user then user can log into the system.
