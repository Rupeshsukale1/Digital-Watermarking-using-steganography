Embed File Help

For Embedding File into another file follow the points given below :

  • Go in the Window menu.
  • Click on the Embed File menu.
  • Select the Source File by browsing the system or enter the path of the intended file.
  • Select Destination File by browsing the system or enter the path of the intended file.
  • Select the file (whose content is to be embedded) by browsing the system or enter the path of the intended file.
  • Click on Embed button if you want to embed the file.
  • Click on Reset button if you want to reset the or clear the fields entered by the user.
  • Click on Send button if you want to send the file to the receiver.
  • Click on Close button if you want to close the Embed File dialog box.

If any of the field is left vacant then system will ask the user to give the proper input.
